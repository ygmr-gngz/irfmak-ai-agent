Meta Developer Kurulum Adımları

  1. Meta Developer App Oluştur: developers.facebook.com → My Apps → Create App
  2. App türü: Business seç
  3. Facebook Login & Instagram Graph API ürünlerini ekle
  4. App Settings → Basic'ten App ID ve App Secret al
  5. .env dosyasına ekle:
    META_APP_ID=xxxx
    META_APP_SECRET=xxxx
    META_REDIRECT_URI=https://yourdomain.com/meta/callback
    META_WEBHOOK_VERIFY_TOKEN=irfmak_webhook_2025
  6. Facebook Login → Valid OAuth Redirect URIs'e META_REDIRECT_URI değerini ekle
  7. Bu paneldeki Bağlan butonuna tıkla → OAuth akışı tamamlanır
  8. Webhook: Meta Developer → Webhooks → Instagram → https://yourdomain.com/webhook/instagram ekle
    Verify token: irfmak_webhook_2025
  9. Webhook subscription: messages field'ını seç
  10. Instagram Business hesabının Facebook sayfasına bağlı olduğundan emin ol

Supabase CRM Tablosu SQL

Supabase SQL Editor'da çalıştır:

CREATE TABLE IF NOT EXISTS crm_leads (
  id UUID PRIMARY KEY,
  source TEXT,
  platform_user_id TEXT,
  customer_name TEXT,
  customer_phone TEXT,
  message TEXT,
  auto_replied BOOLEAN DEFAULT false,
  keywords_matched TEXT,
  handoff_reason TEXT,
  status TEXT DEFAULT 'new',
  created_at TIMESTAMPTZ DEFAULT NOW()
);
